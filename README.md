# 🏡 Lee's House Task Manager

A simple and organized way to track, manage, and complete household tasks for **Lee's new house**. This project helps coordinate chores, prioritize work, and keep everything running smoothly during move-in and beyond.

---

## 📋 Overview

Lee's House Task Manager is designed to:

* Track household tasks
* Assign responsibilities
* Set priorities and deadlines
* Monitor progress
* Keep everything organized in one place

Whether it's unpacking boxes, fixing appliances, or planning future improvements, this project keeps everything structured and easy to follow.

---

## ✨ Features

* ✅ Task creation and tracking
* 🏷️ Priority levels (High, Medium, Low)
* 👤 Task assignments
* 📅 Due dates
* ✔️ Completion tracking
* 📝 Notes and descriptions
* 🔄 Status updates (To Do, In Progress, Done)

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/yourusername/lees-house-task-manager.git
```

2. Navigate to project directory

```bash
cd lees-house-task-manager
```

3. Start adding tasks and organizing work

---

## 🧠 Suggested Workflow

1. Add tasks as they come up
2. Assign priorities
3. Set deadlines if needed
4. Mark tasks as "In Progress"
5. Mark complete when finished

---

## 📌 Task Status

| Status      | Meaning                   |
| ----------- | ------------------------- |
| To Do       | Task not started          |
| In Progress | Currently being worked on |
| Done        | Task completed            |
| Blocked     | Waiting on something      |
